# 支持

[English](SUPPORT.md) | 简体中文

## 获取帮助

- 已确认的缺陷和明确的功能请求，请使用 GitHub Issues。
- 使用问题和部署帮助，如果仓库启用了 GitHub Discussions，优先在那里提问。
- 优先查看根目录 `README.zh-CN.md`，了解安装、Docker Compose 部署和 agent 接入流程。
- 如果你使用的是手动安装或非默认部署方式，请查看 `docs/advanced-install.zh-CN.md`。

## 建议附带的信息

请尽量提供：

- 你的部署方式
- 平台和架构
- 相关日志或截图
- 你实际执行的命令
- 预期结果与实际结果

## 安全问题

不要在公开 Issue 中报告漏洞。请改为遵循 `SECURITY.md` 中的流程。

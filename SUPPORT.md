# Support

## Getting Help

- Use GitHub Issues for confirmed bugs and concrete feature requests.
- Use GitHub Discussions, if enabled, for usage questions and deployment help.
- Check the root `README.md` first for setup, Docker Compose deployment, and agent installation.

## What to Include

Please share:

- your deployment method
- your platform and architecture
- relevant logs or screenshots
- the exact command you ran
- what you expected versus what happened

## Security Issues

Do not report vulnerabilities in public issues. Follow `SECURITY.md` instead.
